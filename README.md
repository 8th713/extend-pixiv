Extend pixiv
=====

Mini userscript for pixiv.

## Features and Usage
visit [pages](https://8th713.github.io/extend-pixiv/).

## Develop
``` sh
npm install
```

To watch and auto-build during development:
``` sh
npm start
```

To build document:
``` sh
npm build
```
