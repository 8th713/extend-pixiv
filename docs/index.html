Extend for pixiv

pixiv 用の各種ユーザースクリプト

About

Google Chrome + Tampermonkey 環境で実行される UserScript です。

準備

  1. Google Chrome をインストールします。
  2. Tampermonkey をインストールします。

注意

  • スマートフォン、Mozilla Firefox + Greasemonkey 環境での動作確認は行っていません。
  • pixiv は、ピクシブ株式会社の登録商標です。
  • このページで配布されているユーザースクリプトはピクシブ株式会社が作成、配布しているアプリケーションではありません。
